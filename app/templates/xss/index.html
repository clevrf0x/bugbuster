XSS Vulnerabilities

Explore different types of Cross-Site Scripting (XSS) vulnerabilities and learn how to exploit and prevent them.

Stored XSS

Level 1: Basic

No sanitization. Use any XSS payload to exploit.

Level 2: Partial Sanitization

Filters <script>. Try other tags or events.

Level 3: Advanced Sanitization

Blacklists common events. Use niche events to bypass.

Reflected XSS

Level 1: Basic

No sanitization. Use any XSS payload.

Level 2: Partial Sanitization

Filters <script> and javascript:. Try other tags or events.

Level 3: Context-Based

Reflected in an attribute. Escape the context.

DOM XSS

Level 1: Basic

Direct DOM manipulation from URL parameter. No sanitization. Easy to exploit.

Level 2: Link Injection

Inject JavaScript into a link. Can you bypass the restrictions?

Level 3: Eval Exploitation

Break out of an eval() function. Advanced challenge with flawed sanitization.
